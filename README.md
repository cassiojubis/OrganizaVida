# OrganizaVida

OrganizaVida é uma aplicação web simples para organizar sua rotina diária e controlar suas finanças pessoais. Permite gerenciar tarefas, salários, contas a pagar e visualizar um resumo geral.

## Funcionalidades

- **Rotina**: Adicione e organize tarefas do dia a dia com horários.
- **Salário**: Registre seu salário mensal e renda extra.
- **Contas**: Gerencie contas a pagar com nome, valor e data de vencimento.
- **Resumo**: Visualize um panorama geral das finanças, contas pendentes e tarefas.

## Como Usar

1. Abra o arquivo `home.html` em um navegador web.
2. Navegue pelas seções usando os botões no menu principal.
3. Adicione dados nas respectivas páginas (Rotina, Salário, Contas).
4. Acesse o Resumo para ver uma visão geral.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage para armazenamento local

## Estrutura do Projeto

- `home.html` / `home.css`: Página inicial
- `rotina.html` / `rotina.js`: Gerenciamento de rotina
- `salario.html` / `salario.js`: Controle de salário
- `contas.html` / `contas.js`: Gerenciamento de contas
- `resumo.html` / `resumo.css` / `resumo.js`: Resumo geral
- `estilo.css`: Estilos compartilhados

## Requisitos

- Navegador web moderno com suporte a LocalStorage
